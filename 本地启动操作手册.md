## 七、如果提示端口被占用

如果启动时看到类似错误：

```text
EADDRINUSE: address already in use 127.0.0.1:3210
```

说明 `3210` 端口已经有服务在运行。

先查看占用端口的进程：

```powershell
netstat -ano | findstr :3210
```

输出结果最后一列是进程 ID，例如：

```text
TCP    127.0.0.1:3210    0.0.0.0:0    LISTENING    25460
```

然后停止这个进程：

```powershell
taskkill /PID 25460 /F
```

注意：这里的 `25460` 只是示例，需要替换成你电脑上实际查到的进程 ID。

停止后重新启动项目：

```powershell
npm run start -- --hostname 127.0.0.1 --port 3210
```
