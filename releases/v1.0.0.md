# v1.0.0 发布说明

发布日期：2026-06-25

## 版本定位

v1.0.0 是“业务场景拆解 Agent”的首个稳定保留版本，用于本地运行、演示和后续迭代基线。

## 核心交付

- 通用业务场景拆解，不限定财务对账等单一场景。
- 支持模型配置页面，用户可自行配置 API Key。
- 支持 DeepSeek OpenAI 兼容 API 地址。
- 输出结果聚焦：
  - 改造结论。
  - 可改造业务流程链路脑图。
  - 上下文对接链路脑图。
  - 数据流转总链路图。
- 所有脑图均为单图链路展示，超宽内容通过横向滚动查看。
- 页面整体为硬核科技赛博视觉风格。

## 快照说明

源码快照包会保存在：

```text
outputs\business-scenario-agent-v1.0.0-source.zip
```

该快照不包含：

- `node_modules`
- `.next`
- `.env` 或真实密钥
- 临时日志和构建缓存

## 回归验证

该版本保留前需通过：

```powershell
npm run lint
npm run test
npm run build
```
