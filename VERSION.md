# 版本记录

## 当前版本：v1.0.0

版本日期：2026-06-25

项目名称：业务场景拆解 Agent

## v1.0.0 版本范围

该版本作为当前稳定可用版本保留，主要能力包括：

- 支持用户配置模型服务商、API Key、模型名称和 OpenAI 兼容 API 地址。
- 支持 DeepSeek OpenAI 兼容接口调用。
- 支持围绕业务场景进行 AI、RPA、API、规则、工作流和人工控制点拆解。
- 输出结构聚焦三部分：
  1. 当前业务流程是否可以进行 AI 或 RPA 流程改造。
  2. 可改造业务流程链路脑图，并标注 AI、RPA、API、规则、人工等标签。
  3. 上下文对接流程与数据流转总链路图。
- 页面采用硬核科技赛博风格。
- 所有主要展示脑图均保持“一个部分一张图、一条链路展示”，超出容器可横向滚动查看。
- 模型返回结构不完全标准时，系统会尽量基于模型真实返回内容和用户输入转换为页面可渲染结构，不再使用旧财务测试数据替代实时结果。
